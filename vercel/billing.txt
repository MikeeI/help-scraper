
  ▲ vercel billing [options] <command>

  Options:

    ls                   Show all of your credit cards
    add                  Add a new credit card
    rm            [id]   Remove a credit card
    set-default   [id]   Make a credit card your default one

  Options:

    -h, --help                     Output usage information
    -A FILE, --local-config=FILE   Path to the local `vercel.json` file
    -Q DIR, --global-config=DIR    Path to the global `.vercel` directory
    -d, --debug                    Debug mode [off]
    -t TOKEN, --token=TOKEN        Login token
    -S, --scope                    Set a custom scope

  Examples:

  – Add a new credit card (interactively)

      $ vercel billing add
