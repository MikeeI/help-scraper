NAME
       update-agent -

DESCRIPTION

       See also: AWS API Documentation

       See 'aws help' for descriptions of global parameters.

            update-agent
          --agent-arn <value>
          [--name <value>]
          [--cli-input-json <value>]
          [--generate-cli-skeleton <value>]

OPTIONS
       --agent-arn (string)
          The Amazon Resource Name (ARN) of the agent to update.
