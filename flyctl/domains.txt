Manage domains

Usage:
  flyctl domains [command]

Available Commands:
  add         Add a domain
  list        List domains
  register    Register a domain
  show        Show domain

Flags:
  -h, --help   help for domains

Global Flags:
  -t, --access-token string   Fly API Access Token
  -j, --json                  json output
      --verbose               verbose output

Use "flyctl domains [command] --help" for more information about a command.
